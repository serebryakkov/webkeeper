Это Телеграм бот (https://t.me/hostkeeperbot) для мониторинга доступности веб ресурсов.

Как он работает:

1. При добавлении сайта бот проверяет введенный пользователем URL на валидность.

2. Затем, если URL валидный, бот проверяет есть ли он уже в списке пользователя. Если нет, то добавляет его в базу и начинает мониторить на предмет доступности.

3. Каждые 30 минут (в будущем планирую сделать возможность выбора интервала) с помощью HTTP (GET) запроса бот получает код ответа сервера и если код 200, то ищет в HTML установленный мета тег. Если тег найден, то ресурс считается доступным.

В ближайшем будущем планирую реализовать непрерывный мониторинг. Т. к. сейчас, при деплое все ресурсы начинают мониториться заново.

В качестве хостинга используется - Heroku. В качестве БД - PostgreSQL.
